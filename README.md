# CocosCreator-simple-handle-node
# 简介

工作中经常遇到频繁显示、隐藏界面的Node，这高频动作还没有快捷键，只能通过通过鼠标来回点击操作浪费不少精力

本插件作为creator 缺失的快捷键隐藏/显示节点快捷键功能的补充

# 使用方法

安装插件后按 *S* 快捷键快速显示、隐藏选中的Node
